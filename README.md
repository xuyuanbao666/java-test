## 运行环境

- **JDK**: 8+
- **IDE**: VS Code (with Java Extension Pack)

## 快速开始

```bash
# 编译
javac src/Lesson01_Variables.java

# 运行
java -cp src Lesson01_Variables
```

## 学习建议

1. **按顺序学习** — 课程由浅入深，建议从 Lesson 01 开始
2. **动手实践** — 每个示例都可以直接运行，修改参数观察变化
3. **理解注释** — 代码中有详细的中文注释，帮助理解每个知识点
4. **举一反三** — 尝试修改示例代码，实现自己的想法
